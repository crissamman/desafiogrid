# Proyecto de Desafío Grid y Rompecabezas

Este proyecto contiene dos ejercicios de práctica relacionados con el uso de CSS Grid y HTML: el Desafío Grid y el Rompecabezas. Ambos ejercicios están diseñados para ayudar a mejorar las habilidades de diseño web utilizando CSS Grid.

## Características

- **Desafío Grid**: Una interfaz similar a una red social que utiliza CSS Grid para organizar publicaciones, historias y sugerencias de amigos.
- **Rompecabezas**: Un ejercicio que muestra cómo organizar imágenes en una cuadrícula para formar un rompecabezas.

## Tecnologías Utilizadas

- **HTML5**: Estructura del contenido.
- **CSS3**: Estilos y diseño responsivo.
- **CSS Grid**: Para la disposición y organización de elementos en la página.

## Estructura del Proyecto

- **index.html**: Contiene la estructura principal de la página del Desafío Grid.
- **style.css**: Estilos específicos para el Desafío Grid.
- **rompecabezas.html**: Contiene la estructura principal de la página del Rompecabezas.
- **rompecabezas.css**: Estilos específicos para el Rompecabezas.
